# vscode-scip-debugger README

A vscode extension to assist in debugging [SCIP](https://github.com/sourcegraph/scip) indexes

## Features

Annotates and provides additional information on hover for occurrences within an indexed file

![annotations](doc/annotations.mov)

Provides a tree view for navigating the indexed scip file

![tree-view](doc/tree-view.mov)

## Requirements

The [SCIP cli](https://github.com/sourcegraph/scip) must be installed and available to your path in order to run
  - see [this](https://github.com/sourcegraph/scip/releases) for installing the latest version
